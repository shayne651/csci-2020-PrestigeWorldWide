To run this program you will want to open it in intellij and run the serverStart1.java class. After the serverStart1.java is started you can start the clientStart.java, at this point there will be two chat windows open . You can type in the text boxes and press enter to send the message. There are also two buttons, Download and upload when you choose upload it will bring up a fileSelecter and you can choose the file you want to send. Once the file has been chosen on one chat application you just need to click download on the other.
